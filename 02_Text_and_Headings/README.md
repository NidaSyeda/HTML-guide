## 🧩 Understanding Headings

HTML provides **six levels of headings**:

```html
<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Smaller Subheading</h3>
<h4>Even smaller...</h4>
<h5>Almost tiny...</h5>
<h6>The smallest heading</h6>
